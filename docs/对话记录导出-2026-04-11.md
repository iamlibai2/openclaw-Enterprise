**问题在于：这个设计有循环！**

- Prometheus Agent 已经在 Gateway 上运行
- 它可以直接调用 Skill
- 但它调用 workflow-execute Skill，这个 Skill 又调用 Backend
- Backend 的 skill_invoker 又试图调用 Gateway

---

**两个可能的设计方向：**

**方案 A：Backend 只做编排，不调用 Skill**
```
Backend WorkflowEngine:
- 只负责拓扑排序、变量解析
- 不调用 Skill，只返回"需要调用哪些 Skill"的信息
- Prometheus Agent 收到结果后，自己调用 Skill 并把结果发给 Backend
```

**方案 B：Backend 是独立执行引擎，Skill 在 Backend 本地执行**
```
Backend WorkflowEngine:
- Skill 调用改为本地执行（直接运行 Python 脚本）
- 不经过 Gateway
- Gateway 只用于 Prometheus Agent 对话和进度推送
```
