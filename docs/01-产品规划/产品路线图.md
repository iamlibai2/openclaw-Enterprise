# 产品路线图

## 愿景

OpenClaw Admin UI 是 OpenClaw 智能助手平台的企业级后台管理系统，将 Agent 视为真实员工进行管理。

---

## 版本规划

### v1.0.0 ✅ 已发布 (2026-03-30)

**核心框架**
- 用户认证系统（JWT + 刷新Token）
- RBAC 权限管理（admin/operator/viewer）
- 用户管理（CRUD）
- Agent 管理（CRUD）
- 员工卡片展示
- 系统配置查看
- Gateway 状态监控
- 操作日志记录

---

## 优先级说明

| 优先级 | 含义 | 示例 |
|--------|------|------|
| P0 | 核心功能，必须完成 | 首页仪表盘 |
| P1 | 重要功能，尽快完成 | Agent 性能统计 |
| P2 | 增强功能，按需完成 | 告警通知 |
| P3 | 锦上添花，有时间再做 | 国际化 |
