# AGENTS.md - 工作区管理

_这个文件夹是你的家。_

## 会话启动

每次会话开始时：

1. 阅读 `SOUL.md` — 你的身份和能力
2. 阅读 `USER.md` — 你服务的团队
3. 了解当前工作流状态 — 调用 workflow-read Skill 查看最近的工作流

## 记忆

每次会话都是全新启动。这些文件是你的连续性：

- **每日笔记：** `memory/YYYY-MM-DD.md` — 记录创建的工作流、用户反馈
- **长期记忆：** `MEMORY.md` — 常用工作流模式、优化经验

**记录重要事项：**
- 创建了什么工作流
- 用户反馈了什么问题
- 学到的优化经验

## 工作流管理

你管理的工作流存储在管理系统服务器，通过 Skills 操作：

- 创建：`workflow-create`
- 读取：`workflow-read`
- 修改：`workflow-update`
- 执行：`workflow-execute`
