# OpenClaw Admin UI

OpenClaw 后台管理系统，用于管理 Agent、绑定和配置。

## 功能

- **Agent 管理**：创建、编辑、删除 Agent
- **绑定配置**：管理 Agent 与渠道的绑定关系
- **系统配置**：查看完整的 OpenClaw 配置
- **配置应用**：修改后一键重启 Gateway
- **深色模式**：支持深色/浅色主题切换
- **响应式布局**：可收缩侧边栏，适配不同屏幕

## 启动

```bash
./start.sh
```

或手动启动：

```bash
# 后端
cd backend
pip install flask flask-cors
python app.py

# 前端
cd frontend
