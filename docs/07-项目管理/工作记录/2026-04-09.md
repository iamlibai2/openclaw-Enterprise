## 技术要点

### 火山引擎图片 URL 有效期

```
URL 签名参数:
  X-Tos-Expires: 86400 秒 (24小时)
  X-Tos-Date: 请求时间
  X-Tos-Signature: 签名
```

解决方案：生成时立即下载到本地服务器，保存本地 URL。

### 事件监听机制

```
用户 ↔ 前端 ↔ Gateway ↔ Agent
                ↓
         WebSocket 事件
                ↓
           后端监听器 (events/listener.py)
                ↓
           事件路由器 (events/router.py)
                ↓
         判断是否发朋友圈
                ↓
         LLM 判断 + 图片生成
```
