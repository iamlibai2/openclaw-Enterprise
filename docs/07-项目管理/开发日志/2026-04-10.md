# 开发日志 - 2026-04-10

## 今日工作

### 1. Gateway Agent API 调研

**发现**：

```javascript
// Gateway 支持的 Agent API
"agents.list"      // 获取 Agent 列表
"agents.create"    // 创建 Agent（会重启）
"agents.update"    // 更新 Agent（热加载）
"agents.delete"    // 删除 Agent（热加载）
"agents.files.get" // 获取 Agent 文件
"agents.files.set" // 设置 Agent 文件
```

**关键限制**：

```python
# agents.create 只接受两个参数
sync_call('agents.create', {
    'name': 'MyAgent',           # 必填
    'workspace': '~/.openclaw/workspace-myagent'  # 必填
})
# 不支持 id、model 等参数

# agents.update 的 model 必须是字符串
sync_call('agents.update', {
    'agentId': 'myagent',
    'model': 'bailian/glm-5'     # 字符串，不是 {"primary": "xxx"}
})
```

### 2. Agent 管理重构

**创建流程**：

```python
# Step 1: 创建 Agent（只能指定 name + workspace）
result = sync_call('agents.create', {
    'name': name,
    'workspace': f'~/.openclaw/workspace-{agent_id}'
})

# Step 2: 设置 model（创建后单独调用）
if model_id:
    sync_call('agents.update', {
        'agentId': result['agentId'],
        'model': model_id  # 字符串
    })
```

**编辑流程**：

```python
# 直接调用 agents.update，热加载无需重启
sync_call('agents.update', {
    'agentId': agent_id,
    'name': name,
    'model': model_id  # 字符串
})
# 耗时 0.3 秒（之前用 config.apply 需要 20+ 秒）
```

### 3. 模型管理 API 调研

**结论：模型没有独立 CRUD API**

```python
# 尝试调用
sync_call('models.create', {...})  # ❌ unknown method
sync_call('models.update', {...})  # ❌ unknown method
sync_call('models.delete', {...})  # ❌ unknown method

# 只有 list
sync_call('models.list')  # ✅ 返回模型列表
```

**模型只能通过配置管理**：

```python
config = sync_call('config.get')
# 修改 config['models']['providers'][provider_id]['models']
sync_call('config.apply', {
    'raw': json5.dumps(config),
    'baseHash': config['hash']
})
# 自动触发 Gateway 重启
```

### 4. 等待 Gateway 重启

```python
def _wait_for_gateway_restart(start_time: float, timeout: float = 60.0):
    """等待 Gateway 重启完成"""
    while time.time() - start_time < timeout:
        try:
            sync_call('config.get')  # 能连通说明重启完成
            elapsed = time.time() - start_time
            return True, elapsed, None
        except:
            time.sleep(1)
    return False, timeout, 'Gateway 重启超时'
```

### 6. 模型名称唯一性校验

**动态校验规则**：

```typescript
const formRules = computed<FormRules>(() => ({
  ...baseRules,
  model_name: [
    ...baseRules.model_name,
    {
      validator: (rule, value, callback) => {
        // 编辑模式：原模型名不检查
        if (isEdit.value && currentModel.value?.model_name === value) {
          callback()
          return
        }
        // 检查是否已存在
        const exists = models.value.some(m => m.model_name === value)
        if (exists) {
          callback(new Error('模型名称已存在，请使用其他名称'))
        } else {
          callback()
        }
      },
      trigger: 'blur'
    }
  ]
}))
```

## 技术要点

### Gateway API 对比

| 功能 | Agent API | Model API |
|------|-----------|-----------|
| List | `agents.list` ✅ | `models.list` ✅ |
| Create | `agents.create` ✅ | ❌ 无 |
| Update | `agents.update` ✅ | ❌ 无 |
| Delete | `agents.delete` ✅ | ❌ 无 |

### 浏览器自动填充问题

**原因**：`type="password"` 触发浏览器自动填充登录凭据

**解决**：

```vue
<el-input autocomplete="off" />              <!-- 普通输入框 -->
<el-input type="password" autocomplete="new-password" />  <!-- 密码输入框 -->
```
