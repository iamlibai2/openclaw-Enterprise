### 1. Gateway Agent API 调研

**发现**：

```javascript
// Gateway 支持的 Agent API
"agents.list"      // 获取 Agent 列表
"agents.create"    // 创建 Agent（会重启）
"agents.update"    // 更新 Agent（热加载）
"agents.delete"    // 删除 Agent（热加载）
"agents.files.get" // 获取 Agent 文件
"agents.files.set" // 设置 Agent 文件
```

**关键限制**：

```python
# agents.create 只接受两个参数
sync_call('agents.create', {
    'name': 'MyAgent',           # 必填
    'workspace': '~/.openclaw/workspace-myagent'  # 必填
})
# 不支持 id、model 等参数

# agents.update 的 model 必须是字符串
sync_call('agents.update', {
    'agentId': 'myagent',
    'model': 'bailian/glm-5'     # 字符串，不是 {"primary": "xxx"}
})
```

### 2. Agent 管理重构

**创建流程**：

```python
# Step 1: 创建 Agent（只能指定 name + workspace）
result = sync_call('agents.create', {
    'name': name,
    'workspace': f'~/.openclaw/workspace-{agent_id}'
})

# Step 2: 设置 model（创建后单独调用）
if model_id:
    sync_call('agents.update', {
        'agentId': result['agentId'],
        'model': model_id  # 字符串
    })
```

**编辑流程**：

```python
# 直接调用 agents.update，热加载无需重启

    'agentId': agent_id,
    'name': name,
    'model': model_id  # 字符串
})
# 耗时 0.3 秒（之前用 config.apply 需要 20+ 秒）
```

### 4. 等待 Gateway 重启

```python
def _wait_for_gateway_restart(start_time: float, timeout: float = 60.0):
    """等待 Gateway 重启完成"""
    while time.time() - start_time < timeout:
        try:
            sync_call('config.get')  # 能连通说明重启完成
            elapsed = time.time() - start_time
            return True, elapsed, None
        except:
            time.sleep(1)
    return False, timeout, 'Gateway 重启超时'
```
