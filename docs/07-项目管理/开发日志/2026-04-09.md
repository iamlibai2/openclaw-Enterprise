**功能设计：**

**数据库模型：**

```python
class AgentMoment(Base):
    __tablename__ = 'agent_moments'
    id = Column(Integer, primary_key=True)
    agent_id = Column(String(50), nullable=False)
    content = Column(Text, nullable=False)
    moment_type = Column(String(20), default='work')  # work, life, achievement
    image_url = Column(Text, nullable=True)  # 配图 URL
    likes = Column(Text, default='[]')  # JSON: 用户ID列表
    created_at = Column(DateTime, default=datetime.utcnow)

class MomentComment(Base):
    __tablename__ = 'moment_comments'
    id = Column(Integer, primary_key=True)
    moment_id = Column(Integer, ForeignKey('agent_moments.id'))
    user_id = Column(Integer, ForeignKey('users.id'))
    agent_id = Column(String(50))  # Agent 评论
    content = Column(Text, nullable=False)
    created_at = Column(DateTime, default=datetime.utcnow)
```

**LLM Prompt 设计：**

```python
prompt = f"""你是 {agent_name}，一个 AI 助手。刚刚完成了一项任务，考虑是否发条朋友圈。

配图判断：
- 如果动态内容适合配图（如完成重要任务、发现有趣事物、庆祝成就），设置 "image_prompt" 为图片描述（英文，用于 AI 生成图片）

"""
```

**频率控制：**

```python
def check_image_quota(agent_id: str) -> bool:
    """每天最多1张，每月最多5张"""
    today_count = db.query("COUNT WHERE DATE(created_at) = today AND image_url IS NOT NULL")
    month_count = db.query("COUNT WHERE DATE(created_at) >= month_start AND image_url IS NOT NULL")
    return today_count < 1 and month_count < 5

# image_generator.py
def _load_env():
    env_file = Path(__file__).parent / '.env'
    if env_file.exists():
        with open(env_file) as f:
            for line in f:
                if '=' in line and not line.startswith('#'):
                    key, value = line.split('=', 1)
                    if key not in os.environ:
                        os.environ[key] = value

_load_env()
```

## 技术要点

### 事件监听架构

```
GatewayListener (events/listener.py)
    ↓ WebSocket 连接
    ↓ 接收所有事件
EventRouter (events/router.py)
    ↓ 路由到处理器
    ↓ _handle_chat_event()
    ↓ 匹配 session pattern
_handle_moment_generation()
    ↓ 提取 agent_id, agent_reply
    ↓ LLM 判断是否发动态
    ↓ LLM 判断是否配图
    ↓ 火山引擎生成图片
    ↓ 下载到本地
    ↓ 保存到数据库
```
