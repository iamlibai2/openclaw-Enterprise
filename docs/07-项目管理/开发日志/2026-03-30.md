### 6. Bug 修复

#### BUG-001：登录无限刷新循环

**现象**：登录成功后页面无限刷新，无法进入系统

**原因分析**：
1. 路由守卫仅检查 localStorage 是否有 token，未验证有效性
2. API 请求 401 时自动跳转登录页，但登录页未清除旧数据
3. 导致循环：跳转登录 → 检测有 token → 认为已登录 → 跳转首页 → 401 → 跳转登录

**解决方案**：
```typescript
// router/index.ts
if (to.path === '/login') {
  userStore.clear()  // 清除旧认证数据
  return next()
}

// api/index.ts
if (window.location.pathname === '/login') {
  return Promise.reject(error)  // 不重定向
}
```
